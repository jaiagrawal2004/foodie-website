# 🍔 Foodie Website

A modern and responsive food ordering website built using HTML, CSS, and JavaScript. The website provides an attractive user interface for browsing delicious food items and ordering them online.

## 🚀 Features

* Responsive design for all devices
* Attractive homepage with hero section
* Food menu and featured dishes
* Search functionality
* Smooth animations and hover effects
* User-friendly navigation bar
* Mobile-friendly layout
* Modern UI design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
